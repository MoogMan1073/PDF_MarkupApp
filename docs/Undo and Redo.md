# Undo and Redo

Every edit is undoable: creating, moving, resizing, rotating, restyling,
editing text, deleting, and erasing.

- **Undo** — `Ctrl+Z` (also **Edit ▸ Undo**)
- **Redo** — `Ctrl+Shift+Z` (also **Edit ▸ Redo**)

A multi-mark [[Eraser]] drag counts as one undo step. Deleting a comment from
the [[Comments Sidebar]] is undoable too.
