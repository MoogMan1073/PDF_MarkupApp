# DSI Redline — User Manual

Welcome to **DSI Redline**, a fast PDF markup tool for reviewing AutoCAD
Electrical drawing sets, with a built-in **wire-number extraction & export**
engine.

This manual is a small Obsidian-style vault: use the **Pages**, **Tags** and
**Graph** panels on the left, click any `[[link]]` to jump, and click a `#tag`
to see related pages.

## Start here

- [[Getting Started]] — install, open a PDF, save your work
- [[Viewer]] — scroll, zoom, pan, page navigation
- [[Markup Tools]] — highlight, pen, comments, text boxes, shapes
- [[Comments Sidebar]] — find, filter and delete comments
- [[TODO]] — turn comments into a checkable task list
- [[Wire Numbers]] — extract and classify wire labels
- [[Wire Export]] — produce label files for the printer
- [[PDF Tools]] — split, combine, insert, swap, delete, rotate, convert, crop
